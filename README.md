# @yup-es

> es locale for yup

## Installation
```
# yarn
yarn add yup-es

# npm
npm i yup-es
```

## Usage

### JS
```
const yup = require( 'yup' )
const es = require( 'yup-es' )

yup.setLocale( es )
```

### ES6 / Typescript
```
import * as yup from 'yup'
import es from 'yup-es'

yup.setLocale( es )
```
